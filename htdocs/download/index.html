Cache Connectors

Cache connectors enable deepstream to store data in a distributed cache for horizontal scalability and fast read- and write access

  • redis cache connector

    deepstream.io-cache-redis-NPMGithub

  • memcached cache connector

    deepstream.io-cache-memcached-NPMGithub
  • Can’t find a connector for your cache of choice? Read a tutorial on how to write your own.

Storage Connectors

Storage connectors enable deepstream to store data in a database. This allows to use the database's query capabilities and to persist data when deepstream is used with an in-memory only cache like Memcached. Storing data in a database happens outside of the critical path and doesn't affect update performance.

  • rethinkdb storage connector

    deepstream.io-storage-rethinkdb-NPMGithub

  • mongodb storage connector

    deepstream.io-storage-mongodb-NPMGithub
  • Can’t find a connector for your database of choice? Read a tutorial on how to write your own.

Message Connectors

Message connectors enable multiple deepstream nodes to connect to the same message bus and to distribute updates amongst each other, allowing for horizontal scalability

  • direct message connector

    deepstream.io-msg-direct-NPMGithub

  • amqp message connector

    deepstream.io-msg-amqp-NPMGithub

  • redis message connector

    deepstream.io-msg-redis-NPMGithub
  • Can’t find a connector for your messaging system of choice? Read a tutorial on how to write your own.
